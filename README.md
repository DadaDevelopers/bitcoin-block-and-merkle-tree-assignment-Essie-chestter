# Bitcoin Block and Merkle Tree Assignment

## Task 1: Block Inspection
The block was inspected using a blockchain explorer.

Information collected:
- Block Height
- Block Hash
- Previous Block Hash
- Merkle Root
- Number of Transactions
- Timestamp

## Task 2: Merkle Tree Construction

A Merkle tree was constructed using four transaction hashes.

Steps:
1. Pair transactions and hash them.
2. Hash the resulting pairs.
3. The final hash forms the Merkle Root.
